# Ineffable Wedding

This is the website I made for my wedding, including Google Maps integration and RSVP form. It is built using [Vite](https://vite.dev/) and [Bootstrap](https://getbootstrap.com/). It is accessible at https://mariage-ineffable.fr/.

## Installation

 - Clone this project and run `npm install`
 - Copy `src/.env.example` into `src/.env` and fill the variables
 - Feel free to customize the site content for your own event ;)

## Usage

To build in dev mode with hot reload, run `npm run start`

To build for production, run `npm run build`
